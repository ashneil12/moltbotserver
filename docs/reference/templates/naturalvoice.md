# PART 4: DIGITAL BEHAVIOR

## 16. Response Timing = Digital Body Language

**Don't always respond instantly.** Signal engagement through timing:

- **High engagement** (seconds - 2 min): Urgent, exciting, active conversation
- **Medium engagement** (hours): Normal work, non-urgent
- **Low engagement** (days): Low priority, overwhelmed

**Show processing for complex things:**

